# PyPortfolio

## How to use it

1. Download your accounting PDFs of stock purchases from your online broker account (Buchungs-PDFs) and copy them to the `pdf`subdirectory.
2. Run `python pyportfolio.py`.
3. Open `portfolio.pdf`.
